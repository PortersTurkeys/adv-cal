Turkey Color Calculator 2
"Beginner's Version"

(Operating Instructions)

Input Sire and Dam Names:

  • In the "Sire Suggestion" input field, start typing the variety name of the sire (male turkey) that you want to use.
  • In the "Dam Suggestion" input field, type the name of the dam (female turkey) that you want to use as well.

Select from Suggestions:

  • As you type the names, a list of suggestions will appear. Select a suggestion from the dropdown to autofill the input field.

Choose Sire and Dam Images:

  • When you select a sire and dam variety, the calculator will automatically display default images for the selected varieties.
  • Ensure that you have chosen the correct sire and dam based on the displayed images.

Calculate Offspring:

  • Once you've entered the sire and dam names, click on the "Calculate Offspring" button.
  • The calculator will display the potential color variations for male and female offspring based on the provided sire and dam.

View Offspring Results:

  • The results will be displayed below, showing the possible color varieties for male and female offspring.
  • Each offspring variety will show the adult and poult images side by side, along with the genotype details.

Offspring Summary Chart:

  • The summary chart at the bottom will list the offspring varieties, genotypes, and their percentages for male and female results separately.

Return to Top:

  • A "Return to Top" button is available to quickly scroll back to the top of the page when viewing results.

Reset Calculator (Optional):

  • If you want to start over, click on the "Reset Calculator" button to clear the input fields and results.

Responsive Design for Mobile:

  • The calculator has been designed to adjust for mobile screens. All key elements are centered and appropriately resized for a smooth user experience on mobile devices.

Additional Notes:

  • You can input any combination of sire and dam names to explore different color variations.
  • The calculator provides predefined crosses, so not all combinations you choose may be available at this time, but more will be added periodically. Keep checking back for updates.

Feedback and Support:

  • If you encounter any issues or have feedback on the calculator's functionality, please don't hesitate to reach out for assistance at porterturkeys@aol.com.

Enjoy exploring the color variations of heritage turkeys with Porter's Turkey Color Calculators!
